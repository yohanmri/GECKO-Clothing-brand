# Images — place your files here with these exact names

## Required files:

| File name        | Description                              |
|------------------|------------------------------------------|
| `gecko-logo.png` | GECKO gecko lizard + text logo (on dark/orange bg) |
| `hero-woman.png` | Woman athlete with dumbbell (PNG with transparent bg) |

## How to add them:
1. Export the logo from your design file → save as `gecko-logo.png`
2. Export the hero woman PNG (with transparent background) → save as `hero-woman.png`
3. Drop both files in this folder (`src/assets/images/`)
4. Run `npm run dev` — images will appear automatically.
